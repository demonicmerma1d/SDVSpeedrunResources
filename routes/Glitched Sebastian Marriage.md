# Route
## Notes
- Version 1.6.
- Female gender.
- Chinese.

## Overview
The goal here is to marry Sebastian, primarily using mail injection and item injection.

### File Creation
Name the farmer the name in the formatted linked paste and do not skip the introduction.

### Introduction
Skip after Robin says the player name.

### Day 1
- Read the mail one time and run via the backwoods to the mountain. Meet Sebastian in his bed, then teleport home using the Return Scepter.
- Read the mail again and run back, and stand outside Robin's bedroom to speed her up by standing in her way on her way past, and upgrade the house(2nd option on the menu) as she passes the desk.
- Go outside and sleep 3 days using the tent kits.

### Day 4
- Run inside to Sebastian, gift him the bouquet, then the stardrop tea, then the mermaid pendant. 
- Teleport home, and sleep 3 days (move the bed) until the marraige cutscene. Timing ends on the kiss (marriage autosplit).
