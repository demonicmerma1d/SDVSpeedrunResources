# Route
## File setup
- Begin by being on 1.5 (the telephone got added in this version, and 1.6 doesn't allow flag injecting in regular dialogue). 
- Pick any farm type with a chair in the house (so not hilltop). 
- Select Female as your gender.
- Paste in the Player, Farm, and Favorite pastes, and pre-copy the first rename paste.
- Don't skip the intro cutscene.
- Prove that you're not seeded.

## Start
- Escape twice/spam to get to Robin in the intro. The moment she starts speaking, skip.
- Grab a chair from your house on the way out the door (no rush).
- Run into town, and check the Saloon trashcan. If it has a +speed food, you save 5 seconds.
- Wait for Shane to show up, and gift him the Magic Rock Candy.
- Either eat the +speed buff from the trash, or the MRC Shane gave to you.
- Go wait by pierre.

## Post-9am
- Run inside Pierre's shop.
- If you didn't get the +speed buff, run to Pierre and say hi. This gives a MRC to make up for the one you ate.
- Sell this MRC at the counter, and use the Mountain Warp.
- Run to Robin's, and buy (first prompt) the Telephone.
- Run up to the left transition, place the chair facing into the transition, and use the scythe to get out of bounds.
  - An example chair-assisted oob can be found in [this timestamped run](https://youtu.be/rEySBcwmopc?t=199). Note the use of animation canceling to move faster (optimally cancel when the scythe is half swung).
- Run up, run right (straight behind the boulder, might need to include down).
- Run up right to the Witch's Swamp.

## Post-Railroad
- Take the warp to the Swamp, and run to the Henchman. Talk, and then gift the Void Mayo (you can hold the Void Mayo the whole time, just avoid eating it).
- If you're fast, you can run past the henchman and enter the hut first. Take the warp into the Wizard's Basement.
- From there, run to the statue and place down the telephone. 
- Start renaming yourself; delete the existing name, paste in the new name, delete the existing favorite, copy and then paste in the new favorite.
- Call Clint (second button), and while the phone is dialing you can copy the new name. You still need to alt-tab to copy the new favorites each time though.
- Click through the first dialogue, and e/escape out of the next prompt. Rename yourself again; same deletion and repasting.
- Call Clint again, same click through first dialogue and e/escape out the prompt. 
- Use the Beach Warp Clint gives you, and warp to the Beach.
- Run up to the Town entrance.
- Click as fast as you can through the dialogue, and time ends upon holding up the Soda Machine!
