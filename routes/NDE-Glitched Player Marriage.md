# Route
### Notes
- Optimally version 1.3 (for [WSC](/glitches/Window%20Shift%20Cancel.md)) otherwise 1.6 works.
- This is a multiplayer run, have a nearby cabin.

## The Run
1) Setup the player name with the paste.
2) Watch introduction until robin says player name then skip.
3) While player 2 joins run to their cabin, wait by their bed for them to appear. 
4) Give them the wedding ring, sleep until marriage cutscene. (Note: if on 1.3, do not saveskip in multiplayer, it just makes the game crash)
5) Watch the marriage cutscene, if on 1.3 host should be WSCing all the text boxes, for some reason this causes player 2 to skip ahead in the cutscene as well and finish faster.
